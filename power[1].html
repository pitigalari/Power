Description

"Power" is a powerful utility software, an auto shutdown program which is created not only to shutdown but also to do many other specific actions at a specific time.
  • Shutdown
  • Restart
  • Hibernate
  • Standby (Sleep)
  • Lock
  • Log Off
  • Screen Off
  • Screen Server
Don't just waste hours waiting until your computer finishers coping large files, rendering videos, or some other process to shut it down. Use the product "Power" and protect your valuable time.

Minimum System Requirements

  • Microsoft Windows Vista or higher
  • .Net Framework V. 4.0.3*
  • 256 MB of RAM
*This software is created for Windows 8, if you are using an operating system below windows 8 please update .NetFramework to version 4.

Top of Page

Required Permissions

  • Device Information
    • Read system information such as Date, Time, etc.
    • retrieve running processes.
    • Read, Write, modify and delete files.
  • Do Changes to system power settings
    • Shutdown, Restart, Hibernate, Lock workstation, Log Off, Sleep system with and without administrator privileges.
    • Start Screen Server and Screen Off.
  • Other
    • Access web pages through default web browser.
    • Set display timeout
    • Run commands using 'cmd.exe' and 'Run.exe'
    • Run in the background
    • Install shortcuts
